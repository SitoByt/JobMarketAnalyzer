# JobMarketAnalyzer

### Description:

Filter for Job-Offers based on an API-Key,
Save the results insite of an SQLite Database,
Return results in an MarkDown File,

### Info: 

Adzuna API Documentation:
https://developer.adzuna.com/activedocs#/

SQLite Java documentation:
https://www.sqlitetutorial.net/sqlite-java/


### Git Ignore:
All important files and sensitive information are included within the ´.gitignore´,


### SETUP:
1. Get yourself a API Key for a supported provider (Adzuna only for now), 
2. add it to a copy of the ´api_providers.template.json´ and rename it to ´api_providers.json´ (! important - input the correct name)
    - *Important*: Watch out to write the Providername correctly (!)
3. Start the Programm

**Supported Providers:**
    - ´Adzuna´

### HOW TO USE:
After startup, you can select from different options,
    You should start by adding at least one new criteria;
    Then you can start to search for results, which will be put within the ´/output´ folder as an ´.md´ file.
